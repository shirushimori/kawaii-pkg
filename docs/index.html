Kawaii-pkg

One command. Every package manager.

$ curl -sSL https://raw.githubusercontent.com/shirushimori/kawaii-pkg/main/install.sh | bash

Features

Universal Search
Search across all your installed package managers at once. Find where a package lives — AUR, Flatpak, Snap, or official repos. Uses async parallel search with per-manager timeouts for speed.
Smart Install
Find a package in multiple managers? Kawaii-pkg asks you which one to use. One selection, one install command. Supports fuzzy selection when multiple managers have the package.
13 Package Managers
Works with pacman, yay, paru, apt, dnf, yum, zypper, xbps, nix, apk, brew, flatpak, and snap. Auto-detects which managers are installed.
Shortcuts
-s search · -i install · -r remove · -u update · -I info · -l list · -C clean · -d doctor · -H history · -c config · -v version
Doctor
Runs real system health checks. Detects and fixes broken packages with sudo pacman -Dk, syncs databases, checks mirrors and config.
Sudo Support
Automatically asks for sudo when using managers that require it (pacman, apt, dnf, yum, zypper, apk). No manual sudo needed.
Custom Aliases
Set your own command name during install. Use loda -i firefox instead of kawaii -i firefox. Change anytime via config or reinstall.
Built with Rust
Single static binary. No dependencies. No runtime. Copy it anywhere and it just works.
